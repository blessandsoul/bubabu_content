Iteration menu — что можно крутить

1. Module color

Сейчас magenta #FF1F8F. Можно поменять на cyan #5BC0DE, deep magenta-dark #C70066, или black для max scan reliability.

2. Center logo size

Сейчас 28% площади. Уменьшить до 22% = scan надёжнее, Bubabu меньше. Увеличить до 35% = риск scan, Bubabu бросается в глаза. LOGO_RATIO в script.

3. Rainbow style

Сейчас 6 concentric thin arcs. Альтернативы — solid 6-stripe vertical backdrop, soft pastel gradient blob, или single arc above Bubabu. make_rainbow_backdrop().

4. Finder patterns

3 угла сейчас стандартные square outlines. Можно сделать rounded squares / circles / Bubabu-shaped. eye_drawer param. Risk: scan rate.

5. Background

Сейчас solid butter-cream #FFFAEB. Можно добавить gradient → #FFF6CC для warmth (matches ad cover override). Risk: gradient может ухудшить module contrast.

6. White halo

Сейчас halo radius = 62% от Bubabu size. Можно убрать (без halo, прямо на модулях) или увеличить до 80% для cleaner clear-zone. Trade-off vs visual density.

7. Bubabu mood

Сейчас 1.jpeg normal. Можно 2.jpeg alternative pose или 3.jpeg heart-eyes (rare — только для специальных QR кампаний).

8. URL + UTM tracking

Сейчас bare https://bubabu.ge. Можно добавить UTM tags для analytics: ?utm_source=kiosk&utm_medium=qr&utm_campaign=pair01. Увеличит data length = QR денсер.

9. Output format

Сейчас PNG raster 300 DPI. Могу также SVG vector (бесконечно масштабируется, idealen для print) или PDF.

10. Size variants

Сейчас 1110×1110px (~9cm print). Могу батчем сгенерить: kiosk large 12cm / sticker medium 6cm / business card small 3cm. Каждый размер = свой LOGO_RATIO для optimal scan.
