# Print Spec — Kiosk Category Stickers (physical production)

## Dimensions & shape
- **Size:** ~28–34 cm wide × ~8–11 cm tall footprint (die-cut, varies per category within this envelope; Archil's 25–35 cm range).
- **Shape:** **DIE-CUT** — soft cloud-like organic outline with a thick white border, NOT a rounded rectangle. Shape differs per category by design; the white border + palette + Bubabu owl + BUBABU.GE banner unify the set.
- **One sticker per shelf SECTION** — never one per product.
- **Placement:** on the FRONT glass edge of the shelf, not in the middle of the glass, never over the products.

## Material & finish
- **Stock:** white adhesive vinyl (opaque), **kiss-cut along the white die-cut border** (cut the vinyl, not the backing). The white border is part of the rendered artwork.
- **Lamination:** **matte anti-glare** laminate — mandatory. The kiosk has strong LED lighting; gloss will hotspot and kill readability.
- **Adhesive:** removable/repositionable (shelves get re-merchandised seasonally).
- **Bleed:** 3–5 mm. **Safe margin:** keep text 6 mm inside the trim.
- **Resolution:** 300 DPI at final size. Source = the AI raster (the white die-cut border is rendered into the image); the printer kiss-cuts along that white border. If Georgian text mis-rendered, overlay clean Georgian type in an editor before print.
- **Color mode:** CMYK for print. Match the spot accents below as closely as the printer allows; ask for a proof.

## Sequence Print-Check (mandatory pre-mass-print QA — added 2026-05-31)

Before sending to mass print, lay all 8 rendered stickers in a sequence and verify:

| Check | Pass criterion | Fail action |
|-------|----------------|-------------|
| Bubabu visual identity | All 8 Bubabus look identical (same plush form, same pose, same expression, same goggles) | Re-render the drifting sticker with stronger `1.jpeg + 2.jpeg` ref weighting |
| Bubabu rainbow present | All 8 stickers have the small Candy Pop 6-color rainbow arc behind Bubabu | Re-render any sticker missing the rainbow |
| `BUBABU.GE` pill unified | All 8 have IDENTICAL butter-cream pill + magenta text at bottom-center | Re-render any sticker with per-category ribbon color instead of pill |
| Per-category headline color | Each sticker's Georgian headline uses its locked accent hex per the color table above | Re-render any sticker with wrong headline color |
| Per-category left icon | Each sticker shows its category-specific symbol (magnetic balls / bulb / robot / etc) | Re-render any sticker with wrong/missing category icon |
| Die-cut white border | All 8 have the thick glossy white organic border | Re-render any sticker with rectangle/straight frame |
| Georgian text legibility | Mkhedruli lowercase reads cleanly at 1m distance, no Cyrillic look-alikes | Overlay clean Georgian type in editor + re-print |

**Visual diff method:** print 1 sample of each sticker on white vinyl, place all 8 in a 2×4 grid on a wall, step back 2m, scan eye left-to-right across both rows. Bubabu + BUBABU.GE pill must visually disappear into the brand pattern (same in every spot). Per-category headline + icon must be the only differentiators.

**Hold-back rule:** if any single sticker fails any check → HOLD the entire batch. Mixed-quality stickers on the shelf break the brand system Арчил asked for.

## Typography (for the designer / overlay fallback — NOT for the AI prompt body)
- Georgian: clean rounded **mkhedruli** lowercase only — never mtavruli (uppercase) Georgian.
- English subtitle smaller, warm dark grey.
- Benefit words separated by soft round dots `•`.
- Zero Cyrillic anywhere.

## Merchandising rules (do / don't — from Archil, kept)
**Do**
- Keep Bubabu boxes together, not mixed with other products.
- Group by category: impulse together, creative together, STEM/robotics together, magnetic together.
- One shelf = one main category where possible.
- Leave a little empty space between product groups.

**Don't**
- Don't mix Bubabu with random toys on the same shelf.
- Don't crowd the glass with QR codes and signs in front of products.
- Don't put large stickers in the centre of the glass.
- Don't label everything "smart" — fun/impulse items are "Fun Gifts".
- Don't let the kiosk read like a discount toy stall.
