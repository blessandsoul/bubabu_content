# BUBABU.GE, демо маркета подарков (для показа Арчилу)

## Что это
Один статичный HTML-макет главной маркета подарков на домене **BUBABU.GE**, в стиле uncommongoods.com (подарки для всех, по получателю / поводу / коллекциям), где Bubabu (AI-сова) = флагман внутри. Демо-концепт для показа, НЕ продакшн, без бэкенда. Кнопки, корзина, поиск, фильтры = визуальные заглушки.

## Как открыть
Открой `index.html` в Chrome. Нужен интернет: шрифты (Bricolage Grotesque + Noto Sans Georgian, Google Fonts) + иконки (Phosphor CDN). Без интернета структура и текст работают, шрифт/иконки деградируют. Переносить папкой целиком.

## Домен / бренд
- Домен = **BUBABU.GE** (не Bubabuji). Вордмарк везде ЗАГЛАВНЫМИ латиницей (бренд-закон). Проверено: 0 «bubabuji», BUBABU.GE x3, 0 lowercase.
- Маркет подарков живёт на самом Bubabu.ge, бренд = Bubabu, а сова Bubabu = флагман/детская секция внутри (отдельный cyan-бэнд).
- ФЛАГ (твоё решение): значит сам бренд Bubabu идёт «подарки для всех». Это шире 3 доков клиента (там Bubabu-central, дети/семья) и формально против бренд-правила «no gifts for everyone». Ты так решил, строю как просил, просто фиксирую.

## ВАЖНО про язык (черновик)
Весь грузинский, КРОМЕ verbatim, это ЧЕРНОВИК под твою native-проверку (короткие стандартные лейблы, прозу не выдумывал). Verbatim (готово): теглайн `ბავშვების AI მეგობარი`, адреса магазинов. Проверь формулировки или прогони `/lang` перед показом. Кириллицы в грузинском нет, длинного тире нет (проверено).

## Картинки
Реальные фото есть ТОЛЬКО Bubabu + детские товары (других в системе нет). Поэтому:
- Реальные фото: hero-коллаж, флагман-секция Bubabu, грид «популярные саჩуქреби» (8 детских товаров).
- Плитки получателей / поводов / коллекций = иконка+цвет (фото взрослых подарков нет, для маркета это нормальный паттерн).
- Источники: `bubabu-hero.jpg` (маскот), `bubabu-banner.jpg` (постер 400-360), `logo.jpg`, 8 `p-*.jpg`.
- Цены/имена/счётчики = реалистичный пример, не финал. Взрослые товарные фото пока отсутствуют (надо снять/сгенерить для продакшна).

## Дизайн (премиум + нейромаркетинг)
**База (серьёзная, не детская):** кость `#F6F2EB` + уголь `#211F1D` + глубокий петроль `#1B4A4F` (серьёзный родственник Bubabu-cyan, бренд/доверие) + приглушённая глина `#B05E4D` + латунь `#A8833C`. Плитки получателей/категорий/наборов = приглушённые глубокие тона (петроль/слейт/глина/латунь/олива/слива/терракота), НЕ яркий мульти-цвет. Убран детский Candy Pop по правке юзера «опять детские игрушки вайб».
**Нейро-конверсия (von Restorff pop):** один высококонтрастный экшн-оранжевый `--cta:#E2632B` ТОЛЬКО на действиях покупки: «в корзину» (+), primary CTA, счётчик корзины, цена-сделка 360₾ во флагмане (`--cta-light` на тёмном). Зелёный `--trust:#3E7D5A` на доверии (бесплатная доставка, гарантия). Дисциплина: оранжевый изолирован → каждая buy-кнопка выделяется на спокойной базе.
Шрифт: **Noto Sans Georgian везде** (один шрифт на всё, латиница/цифры/₾/вордмарк тоже им). **Signature: «подарочная бирка» (gift-tag)** на карточках.

## UI/UX-переработка (анти-слоп, 2026-06-17)
По правке «ии слоп»: убран eyebrow-спам (13→4), 3 одинаковых ряда пилюль (повод/бюджет/интерес) объединены в ОДИН редакционный блок «სწრაფი არჩევა» (3 колонки-списки), категории = editorial-список (лейбл слева + строки справа, не грид), hero = type-led + арка-фрейм (не коллаж наклонённых карточек), new arrivals = горизонтальный рейл, reviews = фичер-цитата + 2, why = чистая inline-полоса, hairline-границы вместо мягких теней везде, заголовки слева + supporting-строка + «view all». Цель: ни две секции не в одной раскладке.
**Anti-«мёртвый» пас:** палитра подняли из мутно-земляной в насыщенный тёплый jewel (живой teal/coral/gold/green/plum/terracotta, не candy), тёплый фон. ДВИЖЕНИЕ: бегущая промо-строка (marquee, пауза на hover), zoom фото на hover, «дышащие» градиенты на флагмане/сезоне, флоат hero, slide-стрелки. Всё под `prefers-reduced-motion`. Движение видно ТОЛЬКО в браузере (на скриншоте статично).
