# Bubabu Smart Gifts, демо главной (для показа Арчилу)

## Что это
Один HTML-мокап главной страницы, по мотивам дизайна KiwiCo / Wicked Uncle / Fat Brain Toys / MoMA Store / Firebox, но в брендинге Bubabu. Это демо-концепт, чтобы продать визуальную идею. НЕ продакшн-сайт (продакшн = отдельная сборка techbrain/Next).

## Как открыть
Открой `index.html` в Chrome (двойной клик). Нужен интернет: шрифт Noto Sans Georgian (Google Fonts) + иконки (Phosphor CDN). Без интернета структура и текст работают, шрифт/иконки деградируют до системных. Переносить папкой целиком (index.html + assets рядом).

## ВАЖНО про язык (черновик)
Весь грузинский текст, КРОМЕ документированных verbatim-строк, это ЧЕРНОВИК под твою native-проверку. Verbatim (готово): теглайн `ბავშვების AI მეგობარი`, адреса магазинов (`თბილისი მოლი, მე-3 სართული, კვების სივრცეში` / `გალერია თბილისი, მე-3 სართული, ლიფტის მიმდებარედ`). Всё прочее (заголовки секций, имена товаров, кнопки, буллеты) я НЕ финализирую как KA-копирайтер. Проверь формулировки сам или прогони через меня `/lang` перед реальным показом. Кириллицы внутри грузинских слов нет (проверено), длинного тире нет (проверено).
