## WORDMARK LOCKUP (for cover/banner - editor PNG overlay, NOT generator-baked)

A 3-line stacked lockup. Render the image clean (no text), then composite this in CapCut/Photoshop so letters stay crisp (generators glitch Mkhedruli).

```
   ┌─────────────────────────────┐
   │        ბუბაბუ                │  ← LINE 1  hero wordmark (plush)
   │      SMART GIFTS             │  ← LINE 2  descriptor
   │  AI Toys · Educational Gifts │  ← LINE 3  3-sec promise (optional on tight crops)
   │       · Smart Surprises      │
   │         BUBABU.GE            │  ← domain (bottom band)
   └─────────────────────────────┘
```

**LINE 1 - hero wordmark `ბუბაბუ`** (Mkhedruli, the established brand wordmark):
- Plush stuffed-fleece letters: chunky rounded display, slight fuzz halo, navy `#1A1A2E` stitch-seam edges. NOT glossy, NOT chrome, NOT balloon.
- Color: Bubabu Cyan `#5BC0DE` fill (the signature goggle color).
- Font for overlay: **Noto Sans Georgian Bold** (real font, tinted cyan). Mkhedruli only - NO mtavruli/uppercase Georgian.

**LINE 2 - `SMART GIFTS`** (English descriptor - this is what carries the rebrand):
- Clean modern geometric sans, letter-spaced ~8%, all-caps Latin.
- Color: Soft Yellow `#FFD23F` or Cream `#FAF7F2` (pick whichever reads on the cover background).
- Weight: semibold. Sits directly under the owl wordmark, ~45% of its height.

**LINE 3 - `AI Toys · Educational Gifts · Smart Surprises`** (3-sec promise, optional):
- Thin sans, Cream/navy, small. Drop on narrow crops (TikTok/IG profile) where space is tight.

**DOMAIN - `BUBABU.GE`**:
- Clean Latin sans, UPPERCASE, navy `#1A1A2E` or Cream. Bottom band, above any mobile-UI safe zone.
